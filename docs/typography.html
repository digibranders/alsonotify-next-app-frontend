1. Global Foundations

Font Families

Type Family Usage CSS/Tailwind Class
Primary Font Manrope Used globally across the entire application for all text elements. font-manrope / Global default

Font Weights

Weight Value Tailwind Class Typical Usage
Normal 400 font-normal Body text, descriptions, long-form content, inputs placeholder.
Medium 500 font-medium Input text, dropdown items, secondary small badges, inactive tabs.
SemiBold 600 font-semibold Widget headings, standard buttons, emphasized list items, links.
Bold 700 font-bold Page titles, form labels, primary actions, prominent data metrics.

Text Colors Map

Color Hex Value Tailwind Class Usage & Context
Primary Text #111111 text-[#111111] Headings, active list items, input field values, primary data points.
Secondary Text #666666 text-[#666666] Subtitles, descriptive text, "View All" links, standard metadata.
Muted Text / Labels #999999 text-[#999999] Form field uppercase labels, input placeholders, minor context text.
Brand Accent #ff3b3b text-[#ff3b3b] Primary links, active states, important call-to-actions, "Today" modifiers.
White text #ffffff text-white Text inside primary brand buttons or dark overlays.
