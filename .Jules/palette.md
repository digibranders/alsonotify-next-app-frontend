## 2026-03-09 - Accessibility standardization for global banners
**Learning:** Dismissible notification banners require the `role="alert"` and `aria-live="polite"` attributes to ensure screen readers announce them upon rendering. Furthermore, decorative SVG icons embedded within buttons (like `lucide-react` icons) require `aria-hidden="true"` to prevent redundant screen reader announcements, and the button itself must rely on visible text or an explicit `aria-label`.
**Action:** Always apply `role="alert"` and `aria-live` to dynamically rendered banners, ensure pure icon buttons have explicit `aria-label`s, hide purely decorative icons with `aria-hidden="true"`, and maintain keyboard accessibility using the `focus-visible:ring-2` class pattern specific to this apps design system.
